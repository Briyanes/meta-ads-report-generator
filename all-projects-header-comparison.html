üí° Rekomendasi untuk Portofolio Project

Agar Portofolio memiliki header yang konsisten dengan QC Scanner dan Meta Ads:

  • Tambahkan lineHeight: '1.2' pada title
  • Tambahkan lineHeight: '1.4' pada subtitle
  • Tambahkan CSS classes: className="responsive-header-title" dan className="responsive-header-subtitle"
  • Bungkus title/subtitle dengan <Link href="/">
  • Copy responsive styles dari globals.css project lain

Catatan: Perbedaan yang Anda lihat kemungkinan besar disebabkan oleh:
1. Browser default line-height - Portofolio tidak set line-height eksplisit
2. System font rendering - Bisa berbeda antar OS
3. Missing responsive CSS - Portofolio tidak punya CSS classes yang sama
