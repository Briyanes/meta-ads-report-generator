Analisis Perbedaan

Kedua header memiliki CSS yang 100% identik:

  • Title margin-bottom: 4px (sama)
  • Subtitle margin: 0 (sama)
  • Title font-size: 32px (sama)
  • Subtitle font-size: 14px (sama)
  • Font-weight: sama (bold untuk title, 600 untuk "Hadona")

Jika Anda melihat perbedaan visual, kemungkinan penyebabnya:

  1. Browser cache - Tekan Cmd+Shift+R (Mac) atau Ctrl+Shift+R (Windows)
  2. System font rendering - Berbeda antara operating system
  3. Browser zoom level - Pastikan zoom 100%
  4. Line-height default - Bisa dipengaruhi browser stylesheet
